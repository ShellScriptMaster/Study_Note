Python 爬虫基础内容
    协议
        显示哪些内容可以提供给用户进行爬取, 哪些内容爬取存在法律风险
    http协议
        超文本传输协议
    常用请求头信息
        User-Agent
            请求载体的身份标识/ 即浏览器信息
        Connection
            请求完毕后保持连接还是断开连接
    常用响应头信息
        Content-Type
            服务器响应回客户端的数据类型
    https协议
        安全的超文本传输协议
        数据进行加密
    加密方式
        对称密钥加密
            加密数据与密钥同时传输给服务端
        非对称密钥加密
            客户端使用公钥对数据加密后，服务端接收加密数据使用私钥对其进行解密
                1. 创建密钥对
                2. 将公钥发送给客户端
                3. 使用私钥对消息进行加密并发送给服务端
                4. 使用密钥对消息进行解密
        证书密钥加密
            在非堆成密钥加密的基础上配合证书对密钥对的认证
